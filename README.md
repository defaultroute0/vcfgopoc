# VMware Cloud Foundation 9.1 — Proof of Concept

A guide for planning and standing up a VMware Cloud Foundation (VCF) 9.1
proof of concept. Replace the sample `*.example.com` hostnames and `<PLACEHOLDERS>` with values
from your own environment.

## Contents

- **[VCF 9.1 PoC Prerequisites](vcf-9.1-poc-prerequisites.md)** — what to prepare before the PoC:
  internet/proxy, VLANs, MTU, dynamic routing, DNS & NTP, storage, firewall ports/URLs, software &
  depot, jump host, naming, passwords, and the management-services FQDNs.
- **[VCF 9.1 Offline Depot How-To](vcf-9.1-offline-depot-howto.md)** — build a local offline
  software depot with the VCF Download Tool for air-gapped install, upgrade, and patching.
